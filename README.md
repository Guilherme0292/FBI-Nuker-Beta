# Instalação

## Linux, Windows:

```console
git clone https://github.com/user001js/FBI-Nuker-Beta

cd FBI-Nuker-Beta

python -m pip install -r requirements.txt

python3 easy-fbi-nuker.pyc
```

## Android, Termux:

```console
apt update && apt upgrade

pkg install python git

git clone https://github.com/user001js/FBI-Nuker-Beta

cd FBI-Nuker-Beta

python -m pip install -r requirements.txt

python easy-fbi-nuker.pyc
```
